# Survey Design System

Kit de diseño y componentes web para encuestas y páginas de consentimiento del proyecto de Investigación de Videojuegos (UVG).

## Estructura del repo

```
survey-design-system/
│
├── assets/            # Recursos visuales (logos, hero images, iconos)
│   ├── logos/
│   ├── hero/
│   └── icons/
│
├── playground/         # Entorno de pruebas para ver los componentes en acción
│   ├── index.html
│   ├── consent.html
│   ├── css/
│   │   └── style.css
│   └── js/
│
├── components/          # Componentes HTML reutilizables
│   ├── header.html
│   ├── hero.html
│   ├── cards.html
│   ├── project.html
│   ├── consent.html
│   └── footer.html
│
└── branding/            # Guías de marca
    ├── colors.md
    ├── icons.md
    └── typography.md
```

## Uso

Abrí `playground/index.html` en el navegador para ver el playground de componentes.
